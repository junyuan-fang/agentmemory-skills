---
description: Extract reusable skills from a session or date range, write SKILL.md, commit.
---

Distill a slice of conversation history into SKILL.md files. Default: today.

```bash
ccskill extract $ARGUMENTS
```

Common args: `--date 2026-05-14` · `--session s1` · `--since 2026-05-10` · `--query "..."` · `--dry-run`.

If skills were written, mention the slugs and ask whether to push to the remote.
